# Photo assets

The Photos app loads square JPGs from this folder. If a file is missing,
the app falls back to a themed gradient + icon placeholder so the game
still runs while you're sourcing artwork.

Expected files (ids match `PhotosState`):

- `forest_night.jpg` - the clue photo. Dark, ominous, blurry silhouette
  on a forest road. Shot date in-game: "Wczoraj, 23:45".
- `selfie.jpg` - a smiling selfie of the missing girl. Bright, daytime.
- `coffee.jpg` - a flat-lay of a coffee cup at a cafe.
- `cat.jpg` - a cosy photo of a cat at home.

Square crops (1:1) work best with the 3-column grid.
