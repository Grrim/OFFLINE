# Image assets

Drop your images into this folder. Phase 1 expects:

- `lockscreen_wallpaper.jpg` - the photo shown behind the lock screen.
  Suggested: a portrait-oriented photo evoking the missing person
  (e.g. an old polaroid, a forest path, a foggy street).
- `home_wallpaper.jpg` - the wallpaper shown behind the home screen
  app grid. Keep it slightly darker so app labels stay readable.

If a file is missing the app falls back to a dark gradient, so the
game still runs without these assets.
